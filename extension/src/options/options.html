FOC GH

GitHub extension for FOC project management.

Authentication

Connect to the GitHub API with Sign in with GitHub (OAuth) or a personal access token. Token permissions · OAuth setup

Only one method is active at a time. Saving a PAT while in “Personal access token” replaces a prior GitHub sign-in token.

OAuth scopes: repo, read:org, project. Build with GITHUB_OAUTH_CLIENT_ID set (see docs).

Cross-org limitation: adding issues or PRs from an org that restricts third-party OAuth may require org approval of this app, or use a PAT instead. Learn more

Never commit this token. Revoke from GitHub settings when rotating.

Global projects

Add project URLs that you want to be global (one per line). These drive which org projects the extension treats as global (i.e., available in the UI of other organizations). They are not limited to the repo list below.

Organization project links only.

Repos to proactively check for global project membership

Add one owner/repo per line.

Issues/PRs in these repos are are automatically checked for global project membership when they are loaded in the GiHub UI. Wherever they are members, a project widget is rendered for that global project.

Expand all projects in sidebar

Expands GitHub’s right-hand Projects blocks on issues and PRs on page load. This includes any global projects that "proactive check" repos above are members of.

Uncheck to leave initial collapse state to GitHub: status rows stay visible. Selecting this option means project detail sections expand.
