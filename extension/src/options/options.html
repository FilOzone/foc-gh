Authentication

Only one method is active at a time. Saving a PAT while in “Personal access token” replaces a prior GitHub sign-in token.

OAuth requests scopes equivalent to the PAT doc: repo, read:org, project. Build with GITHUB_OAUTH_CLIENT_ID set (see docs).

Cross-org limitation: adding issues or PRs from an org outside FilOzone to a global board requires that org to approve this OAuth app. If you see an "OAuth App access restrictions" error, ask an owner of that org to approve the app, or switch to a Personal access token (PATs are not subject to this restriction).
